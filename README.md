# Short-link
Simple link shortener written in Java.

Used:
- Java 14
- Spring Framework 
  - Spring Boot 2.4.0-M1
  - Spring Data JPA 2.4.0-M1
- Thymeleaf 3.0.11.RELEASE
- JUnit 5.6.2

## How to use:
In command line run
### On Linux/Macos:
`./mvnw spring-boot:run`
### On Windows:
`mvnw.cmd spring-boot:run`

and open http://localhost:8080/ in your browser.
